eunice

Improve cohesion and modularity through hierarchical structure and unidirectional (non-circular) dependencies.

click here to learn how Eunice works

Eunice analyses code and shows if dependencies are unidirectional or circular across the hierarchical structure.

Analyze your JavaScript/TypeScript and C# code:

  • terminal
    ~/my-source-code
    $ npx eunice
    creates a eunice.html file from JavaScript/TypeScript in the current directory
  • terminal
    ~/my-source-code
    $ dotnet tool install --global eunice

    ~/my-source-code
    $ eunice MySolution.sln
    creates a eunice.html file from a C# solution or project
